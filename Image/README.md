# Project Assets and Supporting Documentation

## Overview

This folder contains supporting files, screenshots, exported datasets, and certification evidence related to the **Loan Insights SQL Project**, which was completed as part of the **DataCamp Associate Data Engineer Certification**.

The purpose of these assets is to provide additional context, documentation, and proof of project completion. These materials complement the SQL solutions contained in the main project repository and demonstrate the process followed throughout the project.

## Contents

### Certification

The certification files included in this folder serve as evidence of successfully earning the **DataCamp Associate Data Engineer Certification**. This certification validates practical skills in SQL, data cleaning, data transformation, data analysis, and working with relational databases.

## Project Objectives

The Loan Insights project focused on preparing and analyzing loan-related data for business reporting purposes.

Key objectives included:

1. Cleaning and standardizing client information.
2. Identifying and replacing missing values.
3. Combining data across multiple related tables.
4. Producing aggregated metrics for business analysis.
5. Applying business rules to improve data quality.
