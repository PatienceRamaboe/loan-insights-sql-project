# Loan Insights SQL Project

This project was completed as part of the DataCamp Associate Data Engineer Certification.

## Project Overview

EasyLoan offers personal loans, car loans, and mortgages to clients in the USA, UK, and Canada.

The analytics team required cleaned and transformed data for reporting and analysis.

## Skills Demonstrated

- SQL Data Cleaning
- Data Type Conversion
- String Manipulation
- Missing Value Handling
- CASE Statements
- Table Joins
- Data Aggregation
- Business Rule Implementation

## Database Tables

- client
- loan
- contract
- repayment

## Tasks Completed

### Task 1
Cleaned client data by:
- Converting dates
- Standardizing employment status values
- Standardizing country values

### Task 2
Identified and replaced missing repayment channels using business rules.

### Task 3
Joined multiple tables to analyze loans for US clients using the online contract system.

### Task 4
Calculated average interest rates by loan type and country.
